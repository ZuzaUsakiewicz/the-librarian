## General info

CRUD React App using local storage. You can create book, edit and delete it.

## Technologies

Project created with:

- Create React App
- React
- CSS
- local storage
- pagination

## Setup

To run this project, install it locally using npm:

```bash
$ cd ../the-librarian
$ npm install
$ npm run start
```
